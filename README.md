pepper_virtual
==============

Tools to simulate a Pepper robot:
- pepper_control: used to control a virtual robot with [pepper_gazebo_plugin](https://github.com/ros-naoqi/pepper_virtual/tree/master/pepper_gazebo_plugin) or [pepper_moveit_config](https://github.com/ros-naoqi/pepper_moveit_config).
- pepper_gazebo_plugin: Gazebo plugin for a Pepper robot
